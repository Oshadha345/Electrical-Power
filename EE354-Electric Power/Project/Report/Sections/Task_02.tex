\subsection{Scope of Verification}
Task 2 verifies the custom NR program against PSSE outputs and compares:
\begin{itemize}
  \item numerical accuracy in bus \(|V|\) and angle,
  \item convergence behavior of NR, GS, and FDLF,
  \item likely reasons for observed differences.
\end{itemize}
PSSE data were taken from the uploaded solver logs and bus report file (\texttt{PSSE data.md}, \texttt{Bus Based Report.pdf}).

\subsection{Discussion of Deviations}
The bus-voltage differences between custom NR and PSSE are very small (maximum \(|\Delta V|=6.9\times10^{-5}\) p.u.), and angle differences remain below \(0.04^\circ\). This indicates that the custom Jacobian implementation and mismatch formulation are numerically consistent with established NR practice \cite{tinney1967powerflow,grainger1994power}.

Convergence behavior follows expected algorithm characteristics \cite{stott1974fast,saadat1999power}:
\begin{itemize}
  \item NR and FDLF converged quickly (3--4 iterations),
  \item GS converged but required many more updates (18 iterations),
  \item NR reached the strongest mismatch reduction per iteration.
\end{itemize}

The residual differences come mainly from:
\begin{itemize}
  \item printed rounding in PSSE bus report values (4 decimal place display),
  \item solver stop criteria and internal scaling differences,
  \item floating-point linear-solve details between implementations.
\end{itemize}

These observations agree with standard load-flow references and practical tool behavior in commercial packages \cite{saadat1999power,siemens_psse}.
