\begin{abstract}
This report documents the implementation of a full Newton-Raphson load-flow solver for the IEEE 9-bus system, followed by method-level comparison with PSSE and voltage sensitivity analysis under independent load variations. All derivations, simulation logic, and result discussions are presented in technical form with reproducible tables and figure placeholders linked to the code workflow.
\end{abstract}
